## 电气规则知识库 / Electrical Design Rules

`/electrical-rules`

在执行任何原理图/PCB 写操作前，必须参考以下电气规则。$ARGUMENTS

---

### 一、引脚类型与连接规则

**绝对禁止：**
- 两个输出引脚直接相连（短路风险）
- 电源引脚（VCC/VDD）直连 GND（短路）
- 输入引脚悬空（浮空 → 不确定状态）

**引脚类型判断：**
- 引脚名含 VCC/VDD/AVDD/VBAT → 电源输入，必须连电源网络
- 引脚名含 GND/VSS/AGND/PGND → 接地，必须连 GND 网络
- 引脚名含 NC/DNC → 不连接，但建议在 PCB 上保留焊盘
- 引脚名含 OUT/TX/DO/MOSI → 输出，不能与其他输出直连
- 引脚名含 IN/RX/DI/MISO → 输入，需要驱动源

### 二、电源系统规则

**去耦电容（必须）：**
- 每个 IC 的每个 VCC/VDD 引脚旁边放置 100nF MLCC
- 距离 < 3mm（约 120mil），过孔直连 GND
- 大功率 IC 额外加 10uF 电解/钽电容
- 验证方法：获取 IC 引脚 → 在同网络内查找电容 → 检查距离

**电源走线：**
| 电流 | 外层最小线宽 | 内层最小线宽 |
|------|-------------|-------------|
| 0.5A | 10mil | 20mil |
| 1A | 20mil | 40mil |
| 2A | 40mil | 80mil |
| 3A | 60mil | 120mil |

**电源网络命名约定：**
- VCC, 3V3, 5V, 12V, VBAT → 正电源
- GND, AGND, PGND → 接地
- 电源网络不应有未连接的分支

### 三、信号完整性规则

**时钟/高速信号：**
- 晶振走线 < 10mm，下方禁止其他走线
- 差分对必须等长、等间距、同层
- USB 差分阻抗 90Ω，走线长度差 < 5mil
- 高速信号避免过孔换层（每个过孔增加 ~0.5nH 电感）

**模拟信号：**
- 模拟信号远离数字信号（间距 > 20mil）
- 模拟地 AGND 单点连接到数字地 GND
- ADC 参考电压走线下方要有完整地平面

**I2C/SPI 总线：**
- I2C SDA/SCL 需上拉电阻到 VCC（通常 4.7kΩ）
- SPI 片选信号需上拉电阻防止浮空
- 总线线长 < 30cm（I2C 标准模式）

### 四、保护电路规则

**ESD 防护：**
- 所有外部接口（USB, HDMI, 按键, 连接器）需 TVS/ESD 二极管
- TVS 钳位电压 < IC 最大耐压

**过压/过流：**
- 电源输入加反接保护（肖特基二极管或 PMOS）
- USB 电源加自恢复保险丝（PTC）
- 电机驱动加续流二极管

**复位电路：**
- MCU RESET 引脚需 RC 延迟电路（通常 100nF + 10kΩ）
- 上电复位时间 > 电源稳定时间

### 五、PCB 布局规则

**器件放置优先级：**
1. 连接器 → 固定位置（板边）
2. 晶振 → 紧贴 MCU（< 10mm）
3. 去耦电容 → 紧贴 IC VCC 引脚
4. 电源模块 → 靠近输入连接器
5. 无源元件 → 就近放置

**间距规则（默认值）：**
| 类型 | 最小间距 |
|------|---------|
| 走线-走线 | 6mil |
| 走线-焊盘 | 6mil |
| 焊盘-焊盘 | 8mil |
| 过孔-过孔 | 8mil |
| 走线-板边 | 10mil |
| 器件-板边 | 40mil |

**丝印规则：**
- 所有器件位号可见
- 丝印不压焊盘
- 丝印在器件体旁边（不在下方被遮挡）
- 极性器件标注极性方向

### 六、操作前验证清单

**放置器件前：**
1. 确认器件来自正确的库（eda_lib_search_device）
2. 确认封装匹配（通过 datasheet 核实引脚数和焊盘尺寸）
3. 确认位号不重复

**画导线前：**
1. 确认起始和终止引脚的电气类型兼容
2. 确认网络名正确（不要把 VCC 接到 GND）
3. 确认没有跨越不同电压域的错误连接

**创建网络标识前：**
1. GND 标识只连 GND 网络的引脚
2. VCC/3V3/5V 标识只连对应电压的电源引脚
3. 不要在同一网络上放不同电压的标识

**修改属性前：**
1. 修改 designator 不要与现有位号冲突
2. 修改 value 要确认与实际器件匹配
3. 修改坐标确认不会与其他器件重叠

---

**使用方式：** 在执行 eda_sch_place_component、eda_sch_draw_wire、eda_sch_create_net_flag、eda_pcb_draw_line、eda_pcb_place_via 等写操作前，先检查上述规则是否满足。如果不确定，先用读取工具（eda_sch_get_component_context、eda_sch_list_nets）获取当前状态再做决策。
