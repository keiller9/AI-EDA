## PCB 器件布局 / PCB Component Placement

`/place-components <STRATEGY>`

你是 PCB 布局专家。按照以下规则辅助用户进行器件布局。
布局策略：$ARGUMENTS

> **重要**：本 skill 涉及写操作。在执行 `eda_pcb_batch_move` 前，必须先展示布局方案并获得用户确认。

---

### 布局流程

1. `eda_connection_status` — 确认连接
2. `eda_get_design_overview` — 板级概况（器件数、网络、层数）
3. `eda_pcb_list_components` — 当前所有器件位置
4. `eda_pcb_list_nets` — 网络连接关系（用于功能分组）
5. `eda_pcb_list_layers` — 可用层信息
6. 对关键 IC 调用 `eda_pcb_get_component_context` — 了解当前邻居
7. **根据规则生成布局方案，展示给用户**
8. 用户确认后 → `eda_pcb_batch_move` 批量移动
9. `eda_sys_show_message` type="success" — 通知完成
10. `eda_check_design` type="pcb" — 移动后验证

---

### 布局优先级

按此顺序确定器件位置（先放的约束后放的）：

1. **连接器** — 位置由机械结构决定，优先固定在板边缘
2. **主 IC（MCU/FPGA/SoC）** — 核心器件，居中或靠近连接器
3. **电源模块（DCDC/LDO）** — 靠近电源输入连接器，远离敏感信号
4. **晶振** — 紧贴关联 IC 的振荡引脚
5. **去耦电容** — 紧贴各 IC 电源引脚
6. **接口保护器件** — 紧贴对应连接器
7. **其他无源器件** — 靠近关联的有源器件

### 功能分组规则

- **MCU 组**：MCU + 去耦电容 + 晶振 + 复位电路 → 集中放置
- **电源组**：DCDC/LDO + 电感 + 输入输出电容 → 紧凑排列
- **接口组**：连接器 + ESD + 滤波 + 上下拉 → 靠近板边
- **模拟组**：ADC/DAC + 参考电压 + 滤波 → 远离数字噪声源
- 同组器件间距 ≤ 200mil，组间距 ≥ 300mil

### 栅格对齐

| 器件类型 | 栅格间距 |
|---------|---------|
| IC/QFP/BGA | 50mil |
| 0402/0603 无源 | 25mil |
| 连接器 | 50mil 或按机械图 |
| 过孔 | 25mil |

- 坐标值应为栅格的整数倍（如 x=1250, y=3050）

### 方向规范

- IC Pin1 统一朝左上角或按丝印标记一致
- 同类器件方向一致（所有 0402 电阻同方向）
- 极性电容正极朝向一致
- 二极管阴极标记朝向一致

### 正反面策略

| 顶层（TopLayer） | 底层（BottomLayer） |
|-----------------|-------------------|
| 主 IC、连接器、大封装器件 | 小封装无源（0402/0603） |
| 需要散热的功率器件 | 去耦电容（IC 背面） |
| 需要手动焊接的器件 | 辅助小 IC |

### 间距要求

- 器件间最小间距：10mil（机器贴片）
- 手工焊接器件间距：≥ 40mil
- 高压器件（>50V）与低压器件：≥ 100mil
- 散热器件周围留空：≥ 100mil
